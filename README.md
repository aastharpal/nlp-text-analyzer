# NLP Text Analyzer (MERN Stack)

A full-stack web application built using the MERN stack that analyzes user input text using basic Natural Language Processing (NLP) techniques.
The application performs sentiment detection, word frequency analysis, and stores results in MongoDB using REST APIs.

This project was developed to practice full-stack development, API creation, and basic NLP processing.

---

## Features

* Sentiment analysis of text
* Word frequency counter
* Stopword filtering
* Store analysis results in MongoDB
* RESTful API using Express.js
* Responsive UI built with React
* Clean and simple dashboard

---

# NLP Text Analyzer (MERN Stack)

A full-stack text analysis web application built using the MERN stack.
The application allows users to enter text and perform basic NLP operations such as sentiment detection and word frequency analysis.
The project provides a clean React UI and REST API backend for processing text and storing results in MongoDB.

This project was developed as a practice full-stack application to learn backend APIs, database integration, and basic Natural Language Processing techniques.

---

## Features

* Sentiment analysis of user input text
* Word frequency counter
* Stopword filtering
* Store analysis history in MongoDB
* RESTful API using Express.js
* Responsive UI built with React
* Simple and clean dashboard design

## How to Run Locally

Clone the repository

git clone https://github.com/yourusername/nlp-text-analyzer

Install dependencies

npm install

Run backend

npm start

Run frontend

cd client
npm start
